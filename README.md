
# پروژه: پیاده‌سازی الگوریتم **Quantum Boolean Verification** با Qiskit

این پروژه یک مثال ساده از **محاسبه‌ی حالت کوانتومی و اندازه‌گیری یک رشته باینری** با استفاده از کتابخانه‌ی **Qiskit** است.

## توضیحات

* این برنامه یک رشته‌ی باینری را دریافت می‌کند (مثلاً `"111"`) و یک مدار کوانتومی می‌سازد.
* از **دروازه‌های هادامارد (H)** برای ایجاد **superposition** استفاده می‌شود.
* دروازه‌های **CX** و **Z** برای اعمال عملیات منطقی بر روی بیت‌های کوانتومی استفاده شده‌اند.
* در نهایت، مدار اندازه‌گیری شده و نتیجه‌ی حالت باینری به دست می‌آید.
* **AerSimulator** برای شبیه‌سازی مدار استفاده شده است.

## پیش‌نیازها

* Python >= 3.8
* Qiskit
* matplotlib
* numpy

می‌توانید با دستور زیر کتابخانه‌ها را نصب کنید:

```bash
pip install qiskit matplotlib numpy
```

## ساخت و اجرای مدار

1. رشته‌ی باینری مورد نظر را در متغیر `s` وارد کنید.
2. طول رشته تعیین می‌شود (`n = len(s)`).
3. یک مدار کوانتومی با `n+1` بیت کوانتومی و `n` بیت کلاسیک ساخته می‌شود.
4. عملیات‌ها و اندازه‌گیری روی مدار اعمال می‌شوند.
5. شبیه‌ساز `AerSimulator` برای اجرای مدار استفاده می‌شود.

## نمایش مدار

مدار ساخته شده با `cbv.draw()` رسم می‌شود تا ساختار آن قابل مشاهده باشد.

## اجرای شبیه‌ساز

* شبیه‌ساز یک اجرای مدار (`shot=1`) را انجام می‌دهد و نتیجه‌ی **counts** نمایش داده می‌شود.

## مثال استفاده

```python
s = '111'  # رشته باینری مورد نظر
```

* خروجی شبیه‌سازی، حالت کوانتومی مطابق با رشته‌ی ورودی را به دست می‌دهد.
